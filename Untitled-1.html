1

创建学生表

CREATE TABLE T_STUDENT ( STUNO VARCHAR2(16) PRIMARY KEY, STUNAME VARCHAR2(16) NOT NULL, STUSEX VARCHAR2(6), STUBIR DATE );

关键字解释：

  • CREATE TABLE：创建新表
  • VARCHAR2(16)：可变长度字符串，最大16字符
  • PRIMARY KEY：定义主键，唯一标识每条记录
  • NOT NULL：约束该列不能为空
  • DATE：日期类型，存储出生日期

功能说明：

创建一个名为T_STUDENT的学生表，包含学号、姓名、性别和出生日期四个字段。其中学号是主键，姓名不能为空。

2

查询所有学生数据

SELECT * FROM T_STUDENT;

关键字解释：

  • SELECT：用于从数据库中选择数据
  • *：通配符，表示所有列
  • FROM：指定要查询的表

功能说明：

检索T_STUDENT表中的所有记录和所有列，返回完整的学籍信息。

初学者提示：

这是SQL中最基本的查询语句，适合快速查看表的所有内容。在生产环境中，建议明确指定列名而不是使用*，以提高查询效率和可读性。

10

查询女生姓名

SELECT STUNAME AS "姓名" FROM T_STUDENT WHERE STUSEX = '女';

关键字解释：

  • AS：为列指定别名（显示名称）
  • WHERE：条件过滤，筛选满足条件的记录
  • '女'：字符串常量，需要用单引号包围

功能说明：

从学生表中筛选出性别为"女"的学生，并只返回她们的姓名列，且列标题显示为"姓名"。

初学者提示：

WHERE子句是SQL查询中最重要的部分之一，允许您根据特定条件过滤结果。注意字符串值必须用单引号包围。

26

计算学生总评成绩

SELECT S002_C001_SCORES."期中成绩", S002_C001_SCORES."期末成绩", (S002_C001_SCORES."期中成绩" * 0.4 + S002_C001_SCORES."期末成绩" * 0.6) AS "总评成绩" FROM ( SELECT STUNO, COURSENO, TYPE, SCORE FROM T_SCORE WHERE STUNO = 'S002' AND COURSENO = 'C001' AND TYPE IN ('期中', '期末') PIVOT ( MAX(SCORE) FOR TYPE IN ('期中' AS "期中成绩", '期末' AS "期末成绩") ) S002_C001_SCORES;

关键字解释：

  • PIVOT：行转列操作，将多行数据转换为单行
  • SUBQUERY：嵌套在FROM中的子查询
  • MAX(SCORE)：聚合函数，取最大值
  • AS：为计算列指定别名

功能说明：

查询学号S002在课程C001的期中、期末成绩，并计算总评成绩（期中占40%，期末占60%）。

高级技巧：

PIVOT操作是Oracle特有的功能，用于将行数据转换为列，特别适合需要横向展示多个相关值的场景。

35

查询教授的学生

SELECT DISTINCT s.STUNAME AS "学生姓名" FROM T_TEACHER t INNER JOIN T_COURSE c ON t.TEANO = c.TEANO INNER JOIN T_SCORE sc ON c.COURSENO = sc.COURSENO INNER JOIN T_STUDENT s ON sc.STUNO = s.STUNO WHERE t.TEATITLE = '教授';

关键字解释：

  • DISTINCT：去除重复行
  • INNER JOIN：内连接，基于关联字段合并表
  • ON：指定表连接的条件
  • AS：为列指定别名

功能说明：

查询所有教授教过的学生姓名，通过教师表、课程表、成绩表和学生表的四表连接实现。

多表连接技巧：

多表连接时，建议从核心表开始，逐步添加关联表。使用表别名（如t、c）可以提高查询可读性。

48

教师授课数量统计

SELECT t.TEANAME AS "教师姓名", COUNT(c.COURSENO) AS "讲授课程数量" FROM T_TEACHER t LEFT JOIN T_COURSE c ON t.TEANO = c.TEANO GROUP BY t.TEANO, t.TEANAME ORDER BY "讲授课程数量" DESC, t.TEANAME ASC;

关键字解释：

  • COUNT()：聚合函数，统计行数
  • LEFT JOIN：左连接，包含左表所有记录
  • GROUP BY：分组统计，常与聚合函数一起使用
  • ORDER BY：结果排序，DESC降序，ASC升序

功能说明：

统计每位教师讲授的课程数量，包括没有授课的教师（课程数为0），按授课数量降序、教师姓名升序排列。

初学者提示：

GROUP BY子句中必须包含SELECT列表中的所有非聚合列。LEFT JOIN确保即使没有授课记录的教师也会被包含在结果中。

63

创建员工工资表

CREATE TABLE EMP_SALARY ( EMPNO VARCHAR2(16) PRIMARY KEY, EMPNAME VARCHAR2(16), EMPSEX VARCHAR2(6) DEFAULT '男', EMPBIR DATE, SALARY NUMBER );

关键字解释：

  • CREATE TABLE：创建新表
  • PRIMARY KEY：定义主键约束
  • DEFAULT：指定列的默认值
  • VARCHAR2：可变长度字符串
  • NUMBER：数值类型，适合存储工资

功能说明：

创建一个员工工资表，包含员工编号、姓名、性别、出生日期和工资字段。其中员工编号是主键，性别默认为'男'。

表设计原则：

在创建表时，合理选择数据类型和约束条件非常重要。DEFAULT约束可以减少数据插入时的工作量。

80

计算平均分并存储

DECLARE avascore NUMBER; BEGIN SELECT AVG(SCORE) INTO avascore FROM T_SCORE; DBMS_OUTPUT.PUT_LINE('平均分：' || avascore); END;

关键字解释：

  • DECLARE：声明变量
  • BEGIN：PL/SQL块开始
  • INTO：将查询结果存储到变量
  • DBMS_OUTPUT.PUT_LINE：输出结果
  • END：PL/SQL块结束

功能说明：

使用PL/SQL块计算全校学生的平均分数，并将结果存储在变量中，然后打印输出。

PL/SQL基础：

这是Oracle的存储过程语言，用于编写数据库程序逻辑。在SQL*Plus或SQL Developer中执行前，需要先执行SET SERVEROUTPUT ON;以启用输出。
